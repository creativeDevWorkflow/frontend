# Overflow

- `overflow` control overflow in the box model.
- `text-overflow` clipping individual lines of text.

<br>

## text-overflow

```css
p {
  text-overflow: ellipsis;
  overflow: hidden;
  white-space: nowrap;
}
```

`text-overflow` works for text nodes, for example a `<p>` element.
